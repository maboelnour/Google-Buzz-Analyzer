To run this Tool :
  • Download your buzz data from Google Takout from this link
  • Extract the Downloaded Folder.
  • Open the extracted Folder you will find a folder named "buzz"
  • Copy that folder "buzz" here inside this folder, besides the "Analyze.rb" file.

  • To install Dependencies type in your Terminal after moving to this folder (the terminal will ask for your password):
    • sh install.sh
  • To Run:
    • sudo chmod 777 Analyze.rb
  • Then:
    • ./Analyze.rb
  • Open the file named "index.html" and enjoy with your results :)
